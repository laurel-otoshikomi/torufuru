# TORUFUL 香里園店 - 巻き爪・陥入爪専門サロン

大阪府寝屋川市香里園の巻き爪補正専門店TORUFULのランディングページです。京阪「香里園駅」より徒歩5分。

## 🌟 プロジェクト概要

このプロジェクトは、**toruful.com**のデザインを参考に、**寝屋川市香里園駅周辺での集客**を目的とした、巻き爪・爪トラブル専門店のランディングページです。

### 🎯 ターゲット地域
- **メインエリア**: 寝屋川市香里園周辺
- **対応エリア**: 寝屋川市、枚方市、門真市、守口市

### デザインコンセプト
- **柔らかいブルー系**: 清潔感・安心感を演出
- **親しみやすさ**: 地域密着型のサロンとしての雰囲気

## 🚀 使用方法

### ローカルで表示
1. プロジェクトファイルをダウンロード
2. `index.html` をブラウザで開く
3. インターネット接続が必要（CDN使用）

### カスタマイズ

#### 1. 店舗情報の変更
`index.html` 内の以下を編集：
- 住所（`#access` セクション）
- 電話番号
- 営業時間
- Google Map埋め込みコード

#### 2. 代表者情報の変更
`index.html` の `#staff` セクションを編集：
- 代表者名
- 資格・経歴
- メッセージ内容
- 写真の差し替え（実際の写真に変更）

#### 3. 施術事例の追加
`index.html` の `#beforeafter` セクション：
- Before/After画像の差し替え
- 症状と施術回数の変更
- コメントの編集

#### 4. カラーの変更
`css/style.css` の `:root` セクション
